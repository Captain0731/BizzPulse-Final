## Common Mistake ❌

**DON'T select "Static Site"** - This creates a publish directory error

**DO select "Web Service"** - This runs your Flask app correctly

---

## If You See "Publish Directory" Field

This means you selected the **wrong service type**. 

### Fix:
1. Cancel the current setup
2. Go back to "New +"
3. Select "Web Service" (not Static Site)
4. Publish Directory field should NOT appear
